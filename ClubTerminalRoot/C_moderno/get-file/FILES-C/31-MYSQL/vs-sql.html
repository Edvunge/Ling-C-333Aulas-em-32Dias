Como Conectar ao MySQL com C no Visual Studio

  1. Instale o MySQL conforme os passos descritos nesse link.

Confirme a instalação testando com o comando: CREATE DATABASE terminalroot, pois iremos testar com esse nome de banco de dados. Pra logar no MySQL via CMD, PowerShell ou Windows Terminal, logue: mysql -u root -p"SUA_SENHA" -D "terminalroot" -e "SELECT * FROM info": MeuBanco90@.

  1. Crie um projeto no Visual Studio
  2. Escolha Release e x64
  3. Adicionar > novo item > main.c
  4. Copia o arquivo de teste e altere a senha e o nome do banco de dados conforme vc desejar
  5. Propriedades(x64) > C/C++ > Geral > Diretórios de Inclusão Adicionais > C:\Program Files\MySQL\MySQL Server 8.0\include
  6. Vinculador > Geral > Diretório de Biblioteca Adicionais > C:\Program Files\MySQL\MySQL Server 8.0\lib
  7. Entrada > Dependências Adicionais > libmysql.lib
  8. Copiar o C:\Program Files\MySQL\MySQL Server 8.0\lib\libmysql.dll para dentro do seu projeto.
  9. Depurar para ver se está tudo certo!
